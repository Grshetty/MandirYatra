# MadirYatra Website

A responsive website with a modern navigation bar featuring Home, About, and Our Vision sections.

## Features

### Navigation Bar
- **White background** with professional appearance
- **Logo positioned** on the right side
- **Navigation links** (Home, About, Our Vision) positioned on the right side
- **Orange hover effect** (#FF6B35) on navigation links
- **Smooth underline animation** on hover
- **Fully responsive** design for all devices

### Responsive Design
- **Desktop (1200px+)**: Full navigation with horizontal layout
- **Tablet/iPad (768px-1199px)**: Optimized spacing and font sizes
- **Mobile (480px-767px)**: Hamburger menu with slide-in navigation
- **Small Mobile (<480px)**: Compact layout with adjusted spacing

### Device-Specific Optimizations
- **MacBook Air**: Optimized for 1440px displays
- **iPad**: Perfect layout for 1024px tablets
- **iPhone/Android**: Mobile-first approach with touch-friendly interface

## Color Scheme

- **Navbar Background**: White (#FFFFFF)
- **Text Color**: Black (#000000)
- **Hover Color**: Orange (#FF6B35)

## Interactive Features

- **Smooth scrolling** between sections
- **Mobile hamburger menu** with animated transitions
- **Active link highlighting** based on scroll position
- **Hover effects** with underline animations
- **Logo hover effect** with subtle scale transformation

## Browser Compatibility

- Chrome (latest versions)
- Firefox (latest versions)
- Safari (latest versions)
- Edge (latest versions)
- Mobile browsers (iOS Safari, Chrome Mobile)

## Getting Started

1. Open `index.html` in your web browser
2. Navigate through the sections using the navbar
3. Test responsiveness by resizing your browser window or using developer tools

## Customization

- Replace the placeholder logo in `index.html` with your actual logo
- Modify colors in `style.css` (variables are used for easy customization)
- Add more sections by following the existing pattern in `index.html`
